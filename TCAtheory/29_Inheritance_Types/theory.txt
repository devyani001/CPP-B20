Types of Inheritance
---------------------
1. Single 	  	Inheritance
2. Multilevel 	Inheritance
3. Multiple   	Inheritance
4. Hierarchical Inheritance
5. Hybrid 		Inheritance
--------------------------------------------------------

1. Single Inheritance
   -> B inherits properties of A

2. Multilevel Inheritance
   -> C inherits properties of B, B inherits properties of A

3. Multiple Inheritance
   -> A and B are two parent class, C is child class inherites properties of Both A and B parent Classes.

4. Hierarchical Inheritance
   -> Combination of single + multilevel

5. Hybrid Inheritance
   -> Combination of all Inheritance
---------------------------------------------------------------------------------------------------------------

NOTE :
-------
	There is only single and multilevel. There is no multiple, hierarchical and hybrid inheritance in java.
